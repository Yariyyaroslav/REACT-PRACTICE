# REACT-PRACTICE
1. Checking user status (authorized/unauthorized):
- Display the "Login" button for unauthorized users and the "Logout" button for authorized users.
2. Display of welcome message depending on the time of day:
- If the time is morning, show "Good morning!", if it is afternoon, "Good afternoon!", if it is evening, "Good evening!".
3. Checking the availability of goods in the cart:
- If the product is in the cart, show the "View Cart" button, if the cart is empty, "Cart Empty".
4. Displaying a success or error message after the form:
- If the form is successfully submitted, show the message "Thank you for your feedback!", if an error occurred "An error occurred, please try again."
5. Displaying detailed information based on user role:
- Show the control panel to the administrator, and only basic information to the user.
6. Switching between themes (light/dark theme):
- If the light theme is enabled, apply light styles, if dark, apply dark styles.
7. Displaying the "Download" button for mobile users:
- If the user is logging in from a mobile device, show the "Download app" button, otherwise Hi.
8. Displaying adult content:
- If the user is over 18, show the content, otherwise show an age restriction warning.
9. Display different components based on the selected language:
- If the selected language is English, show the English version of the site, if Ukrainian, show the Ukrainian version.
10. Checking if a product has a discount:
- If the product has a discount, show the discounted price and the corresponding mark, otherwise the regular price.
